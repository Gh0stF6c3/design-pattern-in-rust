# Rust 设计模式汇总
这是一个个人项目，通过学习[Refactoring.Guru](refactoringguru.cn)的设计模式来解决项目开发中实际可能遇到的结构设计问题，为项目提供良好的架构。
> 所有模式设计示例均采用Rust实现，例子来自Refactoring，本项目仅作于示例学习和汇总，方便后续查阅使用
